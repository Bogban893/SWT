# 19. Построение графика работ: матрица зависимостей, сетевой график

### Матрица зависимостей

Таблица, показывающая зависимости между задачами:

**Типы зависимостей:**
1. **Finish-to-Start (FS)** — следующая задача начинается после завершения предыдущей (самая распространённая)
2. **Start-to-Start (SS)** — две задачи начинаются одновременно
3. **Finish-to-Finish (FF)** — две задачи заканчиваются одновременно
4. **Start-to-Finish (SF)** — редкий тип, следующая задача заканчивается при начале предыдущей

### Сетевой график (Network Diagram, PERT Chart)

Визуальное представление задач и их зависимостей:

**Элементы:**
- Узлы (Events) — начало/конец задач
- Рёбра (Activities) — сами работы

**Преимущества:**
- Наглядность
- Возможность увидеть критический путь
- Возможность анализировать влияние задержек

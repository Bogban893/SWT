Бизнес-модель — это система создания, поставки и захвата ценности в компании.

### Компоненты бизнес-модели (Business Model Canvas):

**1. Value Propositions (Ценностное предложение)**
- Какую ценность предоставляет продукт
- Почему клиент должен его выбрать

**2. Customer Segments (Сегменты клиентов)**
- Для кого создаётся продукт
- Разные сегменты могут иметь разные потребности

**3. Channels (Каналы доставки)**
- Как продукт достигает клиентов
- Примеры: интернет, физические магазины, партнёры

**4. Customer Relationships (Отношения с клиентами)**
- Как взаимодействует компания с клиентами
- Примеры: поддержка, сообщество, персональное обслуживание

**5. Revenue Streams (Потоки доходов)**
- Как компания зарабатывает деньги
- Модели: подписка, покупка, лицензирование, реклама

**6. Key Resources (Ключевые ресурсы)**
- Что нужно для создания ценности
- Люди, технология, деньги, партнёры

**7. Key Activities (Ключевые действия)**
- Что нужно делать для создания ценности
- R&D, производство, маркетинг, поддержка

**8. Key Partnerships (Ключевые партнёрства)**
- С кем работает компания
- Стратегические союзы, поставщики

**9. Cost Structure (Структура затрат)**
- Какие затраты необходимы для работы бизнеса
- Зарплата, материалы, маркетинг

### Виды бизнес-моделей:

1. **Subscription-based** — подписка (SaaS)
2. **Pay-per-use** — оплата за использование
3. **Freemium** — бесплатно + премиум версия
4. **Marketplace** — платформа для торговли
5. **Advertising-based** — доход от рекламы
6. **Licensing** — лицензирование ПО

### Диаграмма Гантта

Горизонтальная временная шкала, показывающая задачи и их длительность:

**Элементы:**
- По вертикали — список задач
- По горизонтали — время
- Прямоугольники — длительность каждой задачи
- Стрелки — зависимости между задачами

**Преимущества:**
- Наглядность временной шкалы
- Легко видеть параллельные работы
- Удобна для предоставления отчётов

**Недостатки:**
- Сложнее видеть зависимости (чем на сетевом графике)
- При большом количестве задач становится загромождённой

### Оптимизация ресурсов

Распределение ресурсов так, чтобы:
- Избежать перегрузки людей
- Минимизировать стоимость
- Соблюдать сроки

**Техники оптимизации:**
- Выравнивание ресурсов (Resource Leveling) — сглаживание пиков использования
- Сглаживание ресурсов (Resource Smoothing) — перераспределение при наличии резерва
- Параллелизация работ (где возможно)

### Денежный поток (Cash Flow)

График распределения расходов и доходов по времени:

**Для бюджета проекта:**
- Показывает, когда нужны деньги
- Помогает планировать финансирование
- Выявляет пики расходов

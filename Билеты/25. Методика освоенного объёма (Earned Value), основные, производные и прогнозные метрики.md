### Методология Earned Value (EV)

Интегрированный метод управления проектом, объединяющий расписание, бюджет и объём работ.

### Основные метрики:

**1. Planned Value (PV)** — Плановая стоимость
- Бюджет, выделенный на плановые работы к текущему моменту

**2. Earned Value (EV)** — Освоенная стоимость
- Стоимость фактически выполненных работ (на основе % завершения)

**3. Actual Cost (AC)** — Фактическая стоимость
- Фактически потраченные деньги

### Производные метрики:

**1. Schedule Variance (SV)** = EV - PV
- Если SV > 0, проект впереди графика
- Если SV < 0, проект отстаёт от графика

**2. Cost Variance (CV)** = EV - AC
- Если CV > 0, проект экономит деньги
- Если CV < 0, проект превышает бюджет

**3. Schedule Performance Index (SPI)** = EV / PV
- SPI > 1 — опережение графика
- SPI = 1 — точно по плану
- SPI < 1 — отставание

**4. Cost Performance Index (CPI)** = EV / AC
- CPI > 1 — экономия
- CPI = 1 — точно по бюджету
- CPI < 1 — перерасход

### Прогнозные метрики:

**1. Estimate at Completion (EAC)**
- Прогноз полной стоимости при завершении проекта

**2. Estimate to Complete (ETC)**
- Сколько ещё нужно потратить до конца проекта

**3. Variance at Completion (VAC)**
- Прогноз отклонения по бюджету при завершении
